# IT Asset Tracking System (Project Definition)

A small IT asset tracking system is a relational database designed to manage and monitor organizational IT resources such as laptops, servers, and network devices, including:

* Asset identification (ID, type, serial number)

* Assignment to users

* Location tracking

* Status monitoring (active, in repair, retired)

* Basic inventory reporting

This project simulates such a system using SQL schema design, sample data, and structured queries.

---

## Project Context

This project models a small IT asset tracking system.  
The database is designed to manage IT equipment and its relationship to users and locations.

It supports:

- Tracking assets (laptops, servers, etc.)
- Assigning assets to users
- Monitoring asset status (active, maintenance, retired)
- Generating inventory summaries through SQL queries

All data is synthetic and intended for demonstration purposes.
